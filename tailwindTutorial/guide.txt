2 : Warning : button with lots of css can look messy
what to do? 
a) either make it a component  
b) use directives (base, components, utilities) : these are parts of an approach structuring and organising a style using a configurating file method
    base: applies styles globally
    @layer base{
        h1:{
            color:green;
        }
    }
    components : applies styles on components
    @layer components {
        .card {
            background-color : red;
            border-radius : 3px
        }
    }

    utilities: atomic styles for individual property : margin, padding etc
    @utility flex-center{
        @apply flex justify-center items-center
    }
    @apply : inserts tailwind css into css
    @layer base{
        h3{
            @apply font-medium text-slate-900
        }
    }
    @layer :
    @layer components{
        .card{
            @apply m-10 rounded-lg bg-white
        }
    }
